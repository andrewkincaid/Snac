Welcome to Snac!

Snac is a graphical user interface (GUI) to the scanning software scanimage, a part of the SANE (Scanner Access Now Easy) project.

Snac is developed by me, Fredrik Wallner, and is distributed as a freeware without guaranties. If you use it, and is satisfied or have a problem you are more than welcome to contact me.

While the usage of Snac is quite straightforward, the setup can be kind of tricky and this help file concentrates on that.
